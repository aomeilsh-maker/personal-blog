## 一、动词分类（按国文法/学校文法）

依据语尾的形态以及进行活用时语尾变化方式的不同分为三类（五段，一段，サ変カ変）

| 动词类型 | 特征 | 例词 |
| --- | --- | --- |
| **五段动词** | 语尾是最后一个音 | 書く、読む、話す |
| **一段动词** | 语尾是「いる」、「える」 | 食べる、見る、起きる |
| **サ変動詞** | 「する」及其复合形式 | する、勉強する |
| **カ変動詞** | 「くる」 | くる、持ってくる |

## 二、活用形式

动词有以下活用形式：

| 活用形式 | 说明 | 主要用途 |
| --- | --- | --- |
| 未然形（未然） | 表示未发生的动作 | 接否定、受身、使役、意志等 |
| 连用形（連用） | 连接用法 | 接续ます、て、た等 |
| 连体形（連體） | 修饰名词 | 与终止形同形 |
| 终止形（終止） | 句子结尾 | 基本形/辞书形 |
| 假定形（假定） | 表示假设 | 接续ば等 |
| 命令形（命令） | 表示命令 | 直接使用 |
| 意志形（意志） | 表示意志 | 实际上前面是未然，但单独拿出来 |

## 三、各类动词活用规则

### 1. 五段动词活用

- 特点：五段动词的语尾变话对尾音进行变化，是在尾音所在行进行变化
- 依据活用对应变为 a, i, u, u, e, e, o 段发音

| 活用形 | 变化规则 | 例子（書く） |
| --- | --- | --- |
| 未然形 | 尾音变为a段（接意志时为o段） | 書**か**ない、書**こ**う |
| 连用形 | 尾音变为i段 | 書**き**ます |
| 连体形 | 尾音保持u段 | 書**く**こと |
| 终止形 | 尾音保持u段 | 書**く** |
| 假定形 | 尾音变为e段 | 書**け**ば |
| 命令形 | 尾音变为e段 | 書**け** |
| 意志形 | 尾音变为o段 | 書**こ**う |
- 例如：「書く」是 “か行五段活用”动词
- 五段动词的尾音一定是一个子音而不是母音
- 五段动词的未然形通常为a段，但接意志形助词时变成o段

### 2. 一段动词活用

- 特点：一段动词语尾第一个音不变，最后一个音在ら行变化
- 变化（口语）为-, -, u, u, e, o, -

| 活用形 | 变化规则 | 例子（食べる） |
| --- | --- | --- |
| 未然形 | 去掉る | 食べ**ない** |
| 连用形 | 去掉る | 食べ**ます** |
| 连体形 | 原形不变 | 食べ**る**こと |
| 终止形 | 原形不变 | 食べ**る** |
| 假定形 | る变为れ | 食べ**れ**ば |
| 命令形 | る变为ろ（口语）/よ（书面） | 食べ**ろ**/食べ**よ** |
| 意志形 | る变为よう | 食べ**よう** |
- “-”代表删去最后一个音

### 3. サ変動詞（サ行変格活用）

- 特点：不规则动词
- 尾部的る也是在ら行上变化（口语），和一段一样是为 -, -, u, u, e, o, -

| 活用形 | する形态 | ずる形态 | 例句 |
| --- | --- | --- | --- |
| 未然形 | せ、さ、し | じ、ぜ | しない、される |
| 连用形 | し | じ | します、してください |
| 连体形 | する | ずる | する人、勉強すること |
| 终止形 | する | ずる | 勉強する、存ずる |
| 假定形 | すれ | ずれ | すれば、存ずれば |
| 命令形 | しろ（口语）/せよ（书面） | じろ（口语）/ぜよ（书面） | 勉強しろ/せよ |
| 意志形 | 未然形し＋よう | 未然形じ＋よう | ましょう |
- 复合动词：动作性名词、外来词、拟态语、部分副词和形容词词干加上する/ずる
- ずる的活用与する基本相同，只是把对应的する活用的る前面的さ行假名浊化为ざ行
- 注意：ずる未然形只有じ、ぜ，没有ざ

### 4. カ変動詞（カ行変格活用）

- 只有「くる」一个动词
- 每个活用对应的变形都各不相同（完全不规则）

| 活用形 | くる形态 | 例句 |
| --- | --- | --- |
| 未然形 | こ | こない、こられる |
| 连用形 | き | きます、きて |
| 连体形 | くる | くる人 |
| 终止形 | くる | 学校に来る |
| 假定形 | くれ | くれば |
| 命令形 | こい | こい |
| 意志形 | こよう | こよう |

## 四、各活用形接续的助动词

禁止形事实上不是文法上的活用类型，但它确实算是一种变形，不妨把它和其他活用放在一起

| 活用形 | 可接续的助动词/助词 | 编号 | 例子 |
| --- | --- | --- | --- |
| **未然形** | 否定、受身、使役、意志 | (4) | 書か**ない**、食べ**られる**、させる |
| **连用形** | ます、て、た | (3) | 書き**ます**、食べ**て**、見**た** |
| **連體和終止** | 基本没有特定对应 | (2) | 書く人、食べる |
| **假定形** | ば | (1) | 書け**ば**、食べれ**ば** |
| **命令形** | 本身就是变形 | (0) | 書け、食べろ |
| **禁止形** | 基本型加上な | - | 書くな、食べるな |

## 五、各类动词活用形的特殊情况

### 1. 五段动词特点

- 以う段音结尾的动词（除了る结尾的部分）都是五段动词
- 未然形有两种语尾变化（一般是a段，接意志助词时为o段）
- 五段动词变ない形：将ます前尾音变为あ段加ない（如：知る→知らない）
- 当词尾为「い」时变わない是因为它是わ行变形，当然わ行因为除了わ之外都变为了あ行发音，所以这行的活用也叫做ワア行活用

### 2. 一段动词特点

- 终止形是ます形去ます加る
- いえ段单独的假名加る的是一段动词
- 一段的命令形：口语是ろ，书面是よ

### 3. サ変カ変動詞特点

- ずる可以使用する的活用，但语尾第一个音要浊化
- 也可以变为じる使用一段活用（一般来说都是变为一段来用）
- サ変复合动词：汉字词/外来词/日语词+する
- 复合动词语干表示动作内容，活用时只变する/ずる，语干不变
- 单个的する和くる只有语尾，没有语干

### 4. 动词分类的判断方法

| 动词类型 | 判断规则 | 例词 |
| --- | --- | --- |
| **五段动词** | 终止形以う段音结尾并除去上述一段动词的词 | 書く、読む、話す、泳ぐ |
| **一段动词** | る前为假名时：い段平假名或え段平假名加上る结尾的词 | 食べる、見る、起きる |
|  | る前全部内容只有一个汉字时：部分い段え段假名发音的单音节汉字加上る的词 | 出る、寝る |
| **サ変動詞** | する及复合动词（名词+する） | する、勉強する、練習する |
| **カ変動詞** | くる | くる、持ってくる |

### 5. 特殊情况说明

- いえ段不是单独写成假名而是汉字发音的一部分，且汉字部分为两音节及以上时则是五段动词
- 如果いえ是汉字发音，且汉字就一个音时，可能是一段也可能是五段
- 部分复合动词比如「申し出る」是一段，但他本质上是因为「でる」是一段

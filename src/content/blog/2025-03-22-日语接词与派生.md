## 1. 接词基本概念

| 概念 | 说明 |
| --- | --- |
| 接词定义 | 包括接头词和接尾词 |
| 接尾词功能 | 可以将原来的词派生(转化)为其他词 |
| 接尾词派生范围 | 名词、动词、形容词、形容动词的派生都有 |

## 2. 接尾词词源与派生规律

| 规律 | 说明 |
| --- | --- |
| 主要规律 | 接尾词派生词的词类通常与接尾词本身的词源词类一致 |
| 派生方向 | • 词源为形容词派生为形容词 • 词源为动词派生为动词 • 词源为名词派生为名词 |

## 3. 形容词派生接尾词

| 接尾词 | 词源 | 意义 |
| --- | --- | --- |
| 「~やすい」(易い) | 词源是形容词「易い」 | 容易的 |
| 「~にくい」(難い) | 词源是形容词「難い」 | 困难的 |
| 「~がたい」 | 词源是形容词「堪い/難い」 | 难以的 |
| 「~たい」 | 词源是助动词，表示愿望 | 本质上接近形容词性质 |

### 形容词派生特殊情况：

- 「硬い」かたい
- 「難い」かたい
- 「難い」作为动词连用接尾词表示”难以…“时，浊化为「がたい」

## 4. 动词派生接尾词

| 接尾词 | 词源 | 意义 |
| --- | --- | --- |
| 「~始める」 | 词源是动词「始める」 | 开始 |
| 「~出す」 | 词源是动词「出す」 | 拿出 |
| 「~込む」 | 词源是动词「込む」 | 进入 |
| 「~続ける」 | 词源是动词「続ける」 | 继续 |

## 5. 名词派生接尾词

| 接尾词 | 词源 | 意义 |
| --- | --- | --- |
| 「~物/もの」 | 词源是名词「物」 | 东西 |
| 「~方/かた」 | 词源是名词「方」 | 方式 |
| 「~手/て」 | 词源是名词「手」 | 手，引申为做事的人 |

## 6. 词汇派生系统总结

日语构词法中的一个重要规律是：接尾语派生词的词类通常与接尾语本身的词源词类一致。这个规律表现为： - 名词性接尾语通常派生出名词 - 动词性接尾语通常派生出动词 - 形容词性接尾语通常派生出形容词
